ExtendedConnectX

Description: Constructed a higly customizable Connect Four game.

How to Run Game: using the makefile, type into the terminal "make" to compile the program, and "make run" to run the program.
How to run Test Cases: using the makefile, type into the terminal "make test" to compile the test files and either "make testGB" or
"make testGBMem" to run testcases (both run the same tescases, but one is for the memory efficent implementation , while the other is
for the fast implmentation).
